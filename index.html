Soal SQL (kerjakan semua)
  1. Tulis perintah CREATE TABLE untuk ketiga tabel di atas (lengkapi primary key & foreign key).
  2. Tulis perintah INSERT untuk memasukkan semua data dummy yang diberikan.
  3. Tampilkan semua data dari tabel utama beserta nama kategorinya (gunakan JOIN).
  4. Hitung total pendapatan dari transaksi (jumlah * harga) untuk bulan Maret 2025.
  5. Tampilkan produk yang paling banyak terjual.
  6. Urutkan produk berdasarkan harga termahal ke termurah.
  7. Tampilkan daftar pelanggan (jika ada) beserta total transaksinya (GROUP BY).
  8. Buat query untuk menambah stok produk dengan id tertentu sebanyak 5.
  9. Buat query untuk menghapus data transaksi dengan id tertentu.
  10. Buat query untuk menampilkan produk yang harganya di atas rata-rata harga semua produk.
Tugas Tambahan: Infografis / Poster Edukasi

Buatlah poster edukasi menggunakan Canva (atau gambar tangan) dengan ketentuan:

  • Pilih satu topik dari daftar berikut:
    • Pentingnya basis data dalam kehidupan sehari-hari
    • Perbedaan DDL dan DML beserta contoh perintah
    • Ilustrasi cara kerja JOIN (INNER, LEFT, RIGHT) dalam SQL
    • Penjelasan fungsi agregat (COUNT, SUM, AVG, MIN, MAX)
    • Tips merancang database yang baik (normalisasi sederhana)
  • Poster harus memuat: judul, penjelasan singkat, dan ilustrasi/gambar pendukung.
  • Ukuran kertas A4 (boleh portrait/landscape).
  • Kumpulkan dalam bentuk file PDF/JPEG (jika digital) atau print-out (jika manual).
